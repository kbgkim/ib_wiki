# Inbox (문서 입수처)

이 폴더는 기존에 정리된 문서들이나 외부 자료들을 임시로 저장하고, 위키의 적절한 카테고리로 분류하기 위한 공간입니다.

## 활용 방법
1.  **자료 업로드**: 기존에 보유하신 `.md`, `.txt`, `.pdf` (텍스트 추출 가능 시) 등을 이 폴더에 넣으세요.
2.  **분류 요청**: 저(Antigravity)에게 "Inbox에 있는 문서를 분석해서 위키 구조에 맞게 정리해 줘"라고 명령하세요.
3.  **정리 및 이동**: 내용 분석을 통해 `01_Foundations`, `02_Integrated_IB`, 또는 `03_Assets_Verticals` 하위의 적절한 위치로 문서를 정제하여 이동시킵니다.
