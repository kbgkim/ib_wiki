# 기대손실 (EL; Expected Loss)

## 🔥 목적

기대손실(Expected Loss)은 특정 포지션에서 발생할 것으로 예상되는 통계적인 손실 평균값입니다. 
기업이나 금융기관이 대손충당금을 적립하거나 리스크 자본을 산정하는 가장 기초적인 지표입니다.

### ─────────────

## 📌 개념

기대손실은 부도 시점의 총 노출액(EAD)에 부도가 발생할 확률(PD)과 부도 발생 시 실제 손실률(LGD)을 곱하여 산출합니다.

👉 **EL = EAD × PD × LGD**

### ─────────────

## 🧠 구조 역할

### 리스크 산출 체계
1. 기초 데이터 (Exposure, Collateral) 로드  
2. 리스크 변수 (PD, LGD, EAD) 산출  
3. **최종 기대손실(EL) 계산**  
4. 충당금 적립 및 자본 적정성 판단  

### ─────────────

## 💰 Cashflow 관점

EL은 미래 현금흐름의 **'평균적인 부족분(Expected Shortfall)'**을 의미합니다.

### 손실 전이 구조
미래 현금흐름 시뮬레이션  
→ 시나리오별 부족분 발생  
→ 부족분에 대한 가중 평균  
→ **기대손실(EL) 확정**  

### ─────────────

## ⚖️ 자산별 적용

### PF / ABS / NPL (Credit 자산)
- **방식**: PD × LGD × EAD 표준 모델 적용  
- **특징**: 자금 회수 가능성과 부도 확률이 핵심 변수  

### Equity (지분 자산)
- **방식**: Market Value × Shock (변동성 기반 충격)  
- **특징**: 신용 리스크보다는 가격 변동(Market Risk) 관점에서 EL 산출
